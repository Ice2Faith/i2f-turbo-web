# 单页 markdown 文档站点

- 只需要将markdown文件和资源文件都放到此目录中
- 并在 manifest.json 中配置页面标题和默认首页即可

## 相对路径文档

可以直接写相对路径的文档。

- 示例文件：[hello.md](./pages/hello.md)


## 功能介绍
- 主要是提供一个单页的 markdown 文件查看器
- 可以直接部署在 nginx 等静态网站中
- 也可以直接放到例如 github.io 等的静态网站托管中
- 支持大多数的文本格式的文件查看
- 支持相对路径跳转
- 支持标题导航栏
- 支持切换主题
- 支持搜索标题
- 通过点击左上角logo可以展开/折叠主题导航
